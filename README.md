A simple Python code to manage student records and their grades while applying fundamental concepts like conditions, loops, functions, dictionaries, and file handling..

## Features
- Add new students
- Assign grades to students
- Display student name and results
- Store and retrieve data

## Project Structure
- `main.py` – entry point of the program
- `student.py` – student data structure
- `grades.py` – grade management logic
- `storage.py` – data storage handling
- `display.py` – output and display functions
- `main_menu.py` – user interface and navigation

## How to Run
Make sure Python is installed, then run:

```bash
python main.py
